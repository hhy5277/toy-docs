## 目标
Go语言开发后端API服务，利用跨平台技术Flutter开发移动应用程序，利用Scrapy爬虫框架做数据准备，Docker作为后端服务和爬虫的部署方式，最终Android程序上架Google Play。
**玩具项目，暂定是一个文章阅读，图片浏览类型的咨询类应用程序。**
## 人群
- 有简单编程基础的非程序员
- 想要了解前后端完整链路的初级程序员
- 愿意花精力和时间的任何人，可以没有编程基础
